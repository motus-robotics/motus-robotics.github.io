Overview

The End — Motus2

Motus2 unifies world action modeling, autoregressive world simulation, and value learning into a single egocentric embodied agent — scaling from internet video and human demonstrations to tactile robot trajectories, then continuously improving through self-evolution.

Demo tasks include spray cleaner, screw bulb, cut banana, fry egg, toast bread, multi-finger grasp, and cut asparagus — contact-rich bimanual skills executed on a fully bionic stereo dual-arm dual-hand platform with full-palm tactile sensing.

Architecture

Unified World Model

Multi-token in, multi-token out. Motus2 learns a joint distribution over video, action, and tactile trajectories using a Joint Video-Action-Tactile DiT — unifying World Action Model, Autoregressive World Simulator (AC-WM), and Value Model in a single framework.

Post-training supports four modes: joint, video_idm, action_only, and loop. Sliding-window autoregressive decoding enables long-horizon rollouts at deployment.

Scaling Path

Human Data Pyramid

Motus2 scales along a five-tier human data pyramid — from internet-scale web video, through 110k+ hours of monocular and 20k hours of stereo egocentric data, to human-robot aligned tactile trajectories and target-robot demonstrations.

Human Data Scaling Laws

Increasing human data consistently reduces validation loss. From 2k to 20k hours, models converge faster and reach lower asymptotic error — following a predictable logarithmic scaling law.

L = 0.101 − 0.005 · ln(D)

Self-Evolving

Self-Evolving Agent

Motus2-mbrl closes the loop with model-based RL: a Policy (Joint Model, VLA) interacts with a World Model and Reward Model (Value Model), trained on failure data, suboptimal trajectories, and task-irrelevant data via Ray-based distributed async RL.

Test-Time Scaling

At inference, IDM + FDM iterative denoising alternates action prediction and video rollout — predict action, simulate video, infer action, loop:

Evaluation

Results & ablation

Each component of the Motus2 stack contributes measurably to downstream task performance. On screw-bulb insertion with only 100 teleop demos, the full pipeline lifts success rate from 5% to 90%.
